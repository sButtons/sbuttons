Installation

Install sButttons’s source LESS via npm.

Package managed installs don’t include documentation or our full build scripts. You can also use our npm. 
 npm install sbuttons

sButton CDN

When you only need to include SButtons’s compiled CSS , you can use SButtonCDN.
