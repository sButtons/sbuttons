Installation

Install sButttons’s source LESS via npm.

Package managed installs don’t include documentation or our full build scripts. You can also use our npm. 
 npm install sbuttons

sButton CDN

When you only need to include sButtons’s compiled CSS , you can use SButton CDN.

Explore the docs 
 <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/sbuttons/sbuttons/dist/sbuttons.css">
