How to Use

The CSS file is available for download here.
Download the file and link it as a stylesheet in between your <head> tags, as shown below. 

 <link rel="stylesheet" href="/path/to/sbuttons.min.css">

You can copy the sentence above, and replace the "/path/to/sbuttons.min.css" with the proper path to where you have the file stored.

Additionally, you can also use the buttons by using the CDN. You can directly copy the line below and place it inside your <head> tags, just like you would if you were to download the file. 

 <link rel="stylesheet"href="https://cdn.jsdelivr.net/gh/shahednasser/sbuttons/dist/sbuttons.min.css">
